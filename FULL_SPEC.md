## ADMIN ПАНЕЛЬ (history/barbershop/admin_dashboard.py)

Полностью переписать под новую БД. Текущий путь к БД:
`/Users/ii/AI_Business/GitRepos/barbershop_bot_v2/barbershop.db`

### Вкладки (tabs):
1. **📊 Дашборд** — статистика
2. **📋 Записи** — все записи с фильтрами
3. **⚙️ Управление** — настройки

### Вкладка 1: Дашборд
- Карточки: "Записів сьогодні", "Записів цього тижня", "Скасованих", "Виручка (прогноз)"
- График записей по дням (последние 14 дней) — bar chart
- Топ-3 послуги за тиждень
- Топ-3 майстри за тиждень

### Вкладка 2: Записи
- Таблица со всеми записями (id, клиент, телефон, услуга, мастер, время, статус)
- Фильтры: по статусу (pending/confirmed/cancelled), по мастеру, по дате
- Кнопки действий для каждой записи:
  - ✅ Підтвердити (меняет статус на confirmed, отправляет уведомление клиенту)
  - ❌ Скасувати (меняет статус на cancelled, отправляет уведомление клиенту)
- Выгрузка в CSV

### Вкладка 3: Управление  
- **Заблоковані дати**: показать текущие из business_config.json, добавить/удалить дату
  - При добавлении даты — перезаписать business_config.json
- **Робочі години**: показать start/end, кнопка редактировать
- **Майстри**: список мастеров из конфига

### Уведомления из Streamlit
Для кнопок "Підтвердити" и "Скасувати" — отправлять Telegram сообщение клиенту через Bot API.
Использовать BOT_TOKEN и user_id из записи.
URL: `https://api.telegram.org/bot{TOKEN}/sendMessage`
Использовать requests.post() — синхронно (Streamlit не async).
